# Arrays

## Overview
Dynamic memory allocation in C using malloc, realloc, and calloc.

## Files
- `dynamic_memory.c` - Demonstrates malloc, realloc, and calloc functions

## Key Concepts
- **malloc**: Allocates memory of specified size
- **realloc**: Resizes previously allocated memory
- **calloc**: Allocates and initializes memory to zero
- **free**: Deallocates memory

## Compilation
```bash
gcc -o dynamic_memory dynamic_memory.c
./dynamic_memory
```
