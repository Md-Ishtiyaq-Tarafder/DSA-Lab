# Linked List

## Overview
Implementation of a singly linked list with basic operations.

## Files
- `linked_list.c` - Singly linked list implementation

## Key Concepts
- **Node**: Structure containing data and a pointer to the next node
- **insertAtHead**: Insert element at the beginning
- **insertAtTail**: Insert element at the end
- **deleteNode**: Remove element by value
- **printList**: Display all elements

## Operations Implemented
1. Create Node
2. Insert at Head
3. Insert at Tail
4. Delete Node
5. Display List
6. Free List (deallocate memory)

## Compilation
```bash
gcc -o linked_list linked_list.c
./linked_list
```
